# Campus IA Demo

Este paquete está listo para subirse a GitHub Pages.

## Publicación rápida
1. Creá un repositorio nuevo en GitHub.
2. Subí `index.html` y `.nojekyll` a la rama `main`.
3. En Settings → Pages, elegí `Deploy from a branch` o `GitHub Actions` según tu cuenta.
4. Una vez publicado, la demo quedará accesible desde una URL pública.

## Nota sobre YouTube
Si abrís el HTML como archivo local (`file://`), el reproductor puede mostrar error 153. Publicado por HTTPS suele funcionar correctamente.

## Integraciones incluidas
- YouTube embebido con checkpoint en +1:20 desde el arranque de la demo.
- GeoGebra embebido con una demo simple de vectores.
- JupyterLite REPL embebido para pruebas de Python en navegador.
